# Vortex 8-Ball Pool

A static browser-based 8-ball/9-ball pool game with canvas rendering, local profile storage, cue shop progression, AI opponents, and synthesized billiards audio.

## Play Locally

Run:

```powershell
.\start-game.bat
```

The launcher starts a local server and opens the game in your browser.

## Publish Online

This project is fully static. Deploy the folder contents to any static host:

- Netlify Drop
- Vercel
- Cloudflare Pages
- GitHub Pages

The public entry file is `index.html`.

Deployment pipeline verified from Codex.

You do not need to run `server.ps1` online; it is only for local testing.
